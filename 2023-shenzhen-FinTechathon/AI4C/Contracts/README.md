1\. 账户合约
 
---
 
**1\. 创建链上账户**
###### 为用户生成链上唯一账户

###### URL
> [http://localhost:10100/account/insert](www.api.com/index.php)
 
###### 支持格式
> JSON
 
###### HTTP请求方式
> post
 
###### 请求参数

| 参数       | 必选 | 类型     | 说明         |
| ---------- | ---- | -------- | ------------ |
| user_id    | ture | string   | 用户id       |
| user_money | ture | *big.Int | 用户账户余额 |
| user_name  | ture | string   | 用户名称     |
| user_icon  | ture | string   | 用户头像地址 |
|            |      |          |              |


###### 返回字段
|  返回字段    | 类型     | 说明         |
| ---------- | ---- | -------- |
|tx   |string    |链上地址   |
