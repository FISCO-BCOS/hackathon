# 一、系统环境

## 1. 会计信息系统

### 软件架构

后端：Thinkphp5
前端：LayUi
数据库：MySQL

### 安装条件

Windows/Mac
phpstudy
php-5.5.38+Apache

### 安装教程

1. 启动服务器，设置public为根目录
2. 将 /public/lncms.sql 导入数据库
3. 打开浏览器开始使用


## 2. 同态加密模块

Java + paillier-lib

### 安装条件
jdk1.8.0_241


## 3. 智能合约模块

区块链：FISCO BCOS
智能合约：Solidity

### 安装条件
FISCO BCOS v2.5.0以上
WeBASE v1.4.0以上


## 4. IPFS模块

### 安装条件
Windows/Mac 与会计信息系统和同态加密模块同机部署

### 安装教程
https://docs.ipfs.io/install/command-line/
