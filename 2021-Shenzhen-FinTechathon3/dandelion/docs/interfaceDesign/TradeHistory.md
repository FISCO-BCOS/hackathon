# TradeHistory(交易记录信息)
合约说明：CRUD合约，已继承了Base合约实现其基础的增删改查接口

**参数说明**

| id          | （主键）广告的id |
| ----------- | ---------------- |
| fromAddress | 转账方           |
| value       | 转账的积分数量   |
| timestamp   | 转账的时间       |



## insert

#### 功能说明

* 插入某次交易的记录

#### 接口说明

**入参**

| 参数   | 类型    | 说明                   |
| ------ | ------- | ---------------------- |
| _from  | address | 发起交易的用户地址     |
| _to    | address | 接收积分的广告商的地址 |
| _value | uint    | 交易的数量             |

**出参**

null
